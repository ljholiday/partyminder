# PartyMinder Database Tables

The plugin creates the following custom tables (with the site-specific prefix):

- `partyminder_events`
- `partyminder_guests`
- `partyminder_ai_interactions`
- `partyminder_conversation_topics`
- `partyminder_conversations`
- `partyminder_conversation_replies`
- `partyminder_conversation_follows`
- `partyminder_user_profiles`
- `partyminder_communities`
- `partyminder_community_members`
- `partyminder_community_events`
- `partyminder_member_identities`
- `partyminder_community_invitations`
- `partyminder_at_protocol_sync`
- `partyminder_event_invitations`
